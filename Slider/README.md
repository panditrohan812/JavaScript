# 🖼️ Image Slider

## 📖 Project Description

This Image Slider is a beginner-friendly web project built using HTML, CSS, and JavaScript. It displays multiple images one at a time and allows users to navigate using the Next and Previous buttons. The project also includes automatic image sliding every 4 seconds and basic swipe functionality, allowing users to change images by dragging left or right. This project helps beginners understand DOM manipulation, arrays, event handling, and JavaScript functions.

## 🚀 Features

- ⏭️ Next Image
- ⏮️ Previous Image
- 🔄 Auto Slide (Every 4 Seconds)
- 👆 Swipe / Drag Navigation
- 📱 Responsive Design
- 🎨 Clean and Simple UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript(ES6)
- Bootstrap Icons
